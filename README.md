# 📝 정문업무일지 자동 생성기 (Work Log Automator)

구글 스프레드시트에 입력된 일일 업무 데이터를 읽어와서, 정해진 엑셀 양식(`.xlsx`)의 정확한 위치에 자동으로 채워주는 **Streamlit 웹 애플리케이션**입니다.

## 🚀 주요 기능
* **구글 시트 연동**: 매일 반복되는 데이터를 구글 시트에서 실시간으로 가져옵니다.
* **엑셀 양식 자동화**: 복잡한 셀 좌표를 일일이 찾을 필요 없이 파이썬이 대신 입력합니다.
* **원클릭 생성**: 버튼 하나로 엑셀 파일을 완성하고 즉시 다운로드할 수 있습니다.

## 🛠 사용 방법
1. **구글 시트 준비**: 전용 구글 시트에 데이터를 입력합니다.
2. **공유 설정**: 시트 우측 상단 [공유] 버튼을 눌러 '링크가 있는 모든 사용자'가 '뷰어'로 설정되어 있는지 확인합니다.
3. **웹 앱 접속**: 생성된 Streamlit 웹 주소에 접속합니다.
4. **일지 생성**: 시트 주소를 입력하고 '업무일지 생성하기' 버튼을 누릅니다.
5. **다운로드**: 완성된 엑셀 파일을 확인하고 저장합니다.

## 📂 파일 구조
* `app.py`: 웹 앱 실행을 위한 메인 파이썬 코드
* `requirements.txt`: 앱 구동에 필요한 라이브러리 목록
* `정문업무일지_양식_260211_v2.xlsx`: 기본 엑셀 양식 파일
* `README.md`: 프로젝트 설명서 (현재 파일)

## 🏗 기술 스택
* **Language**: Python
* **Library**: Streamlit, Pandas, Openpyxl
* **Platform**: Streamlit Cloud
